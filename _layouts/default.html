Bankdata 2023 – 2025
Lead Developer · Transactions & Categorisation (Consultant) · Stream-aligned team

Designed and built Bankdata's first production use of public cloud for customer data: an event-driven transaction and categorisation engine on AWS, processing 3.5 billion transactions for 1.4 million customers. Security, scale, and integration with legacy systems all had to land at the same time.

As lead developer on a team of eight, I owned the application architecture and built the solution in Quarkus and Java. CI/CD on GitHub Actions, infrastructure-as-code in Terraform, running on ECS Fargate with SQS/SNS, S3, and Aurora PostgreSQL. The slowest part of the work was the migration: pulling data from on-prem systems into AWS without breaking anything on the security, compliance, or operations side.

The patterns we settled on for secure cloud delivery have since been picked up by other Bankdata teams.
